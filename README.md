# TownFilter

An interactive web application for exploring and filtering towns and cities based on population and geographic coordinates.

## Features

- **Interactive Country Map** - Visual representation of towns across a country
- **Population Filter** - Filter towns by minimum population to highlight specific city sizes
- **Geographic Filter** - Filter towns by longitude and latitude coordinates

## How to Use

1. Open `index.html` in a web browser
2. Use the **Population Filter** slider to show/hide towns based on their population
3. Use the **Geographic Filter** sliders to focus on specific regions:
   - Adjust longitude to filter by East-West position
   - Adjust latitude to filter by North-South position
4. Click the clear buttons to reset individual filters
5. Click presets links for quick filtering options

## Files

- `index.html` - Main application file containing HTML, CSS, and JavaScript

- HTML5
- CSS3
- JavaScript
